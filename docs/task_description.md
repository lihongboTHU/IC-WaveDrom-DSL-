### 任务名称：IC芯片时序图到WaveDrom DSL的逆向转译

### 任务背景与目标： 
随着大语言模型能力的持续提升，工程师可通过将技术文档输入模型，辅助快速理解特定器件及其功能。然而，在集成电路（IC）设计与验证领域，技术文档中往往包含大量时序信号波形图。现有的大语言模型乃至专用的光学字符识别（OCR）模型，普遍难以准确解析此类波形图的语义信息，从而限制了模型对文档内容的整体理解能力。

目前，WaveDrom 规范定义了一种基于 JSON 格式的时序波形描述语言，可视为时序波形领域的领域特定语言（DSL）。若能实现将波形图片逆向转换为符合 WaveDrom 规范的 JSON 描述文件，再将其馈送至大语言模型，模型便可更准确地解析并利用其中的信息。

基于上述背景，本项目的目标是研究并实现“IC芯片时序图到 WaveDrom DSL 的逆向转译”，以弥补现有模型在时序波形识别方面的不足，提升大语言模型在 IC 设计与验证文档理解中的实用性与准确性。

### 输入和输出
**输入：** 一张包含电子信号波形的图像（可能包含数字截图、屏幕拍照、水印干扰或白板手绘等）。
**输出：** 一段符合 WaveDrom 标准语法的 JSON 字符串。该字符串不仅需包含正确的信号名称（OCR能力），还必须精确还原各信号随时间轴（Clock）的电平翻转逻辑（高、低、高阻态、数据有效等，要求一定的逻辑推理与对齐能力）。
